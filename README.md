# 🎨 CraftHeaven

**CraftHeaven** is a beautifully designed, responsive web project built using **HTML**, **CSS**, **Bootstrap**, and **JavaScript**.
It features an interactive user experience with dynamic elements and a functional login page, making it ideal for showcasing craft-based businesses, portfolios, or e-commerce platforms.

## ✨ Features

- 🎨 Clean and modern UI/UX
- 📱 Fully responsive design using Bootstrap
- 🔐 Login functionality (UI)
- 🧭 Navigation bar with smooth scrolling
- 💡 Interactive elements using JavaScript (e.g., modals, toggles)
- 📦 Product or craft showcase layout
- 📩 Contact or newsletter section
- 🔄 Dynamic form validation and feedback (optional)

## 🛠️ Built With

- **HTML5**
- **CSS3**
- **Bootstrap 5**
- **Vanilla JavaScript**
